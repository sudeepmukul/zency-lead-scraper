Hi this is a Lead Scraper lol.
So How will u use it?
Simple ra!

Clone this repo first.
Then download the dependancies.
Run the python code.
It will Ask you 
1. Query -> eg: Cafes in Amravati or Pragati Nagar.
2. How many leads u want it to generate -> lets say 100 leads.
3. You want it to scrape websites? Y/N
4. What format u want the data in? Excel/CSV/Both
5. Start

Thats all
Phase 1 it gathers all the data
Phase 2 it Scrapes contact info
Phase 3 it Scores the lead (if he is a potential client or not)
Phase 4 it saves the data into a new csv file.

Now Open the wow.html in browser and select the lead CSV and you can now easily read the csv , copy the lead details.
